# PHP Exercise - V21.0.5



## Installation

```
composer install

npm install

php artisan serve

```
<b>*This project has Laravel Sail Configured*</b>
## Configuration

- Ensure a valid email driver is configured
- configure <b>RAPID_API_KEY</b> and <b>RAPID_API_HOST</b> environment variables in the <i>.env</i> file
